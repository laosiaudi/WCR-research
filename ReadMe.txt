程序运行方法：因为程序在Mac OS平台完成，因此Executable中的WCR在您的计算机上可能无法运行。所以在 SourceCode下已经为您提供了一个Makefile,请在终端下切换目录到SourceCode后使用make的命令，新的WCR将会覆盖旧的在Executable中的WCR。然后再切换目录到Executable下，使用./WCR facebook-hour.txt facebook-query-1.txt result.txt time.txt 运行即可。
